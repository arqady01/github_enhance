# Overview
这是一个给Readme文件增加目录的浏览器插件

- 关键成本驱动因素

影响本项目总预算的关键因素主要包括：

  - **人力资源成本：** 这是预算构成中最主要的部分，特别是在中国当前竞争激烈的人才市场环境下尤为显著。
  - **功能模块复杂度：** 刷题系统（涉及多种题型、练习模式、结果判定）和后台管理系统（涉及内容管理、用户管理、数据统计等复杂逻辑）的开发工作量较大，是主要的成本贡献模块。
  - **云基础设施配置：** 服务器、数据库等云资源的选择（规格、计费模式）将直接影响基础设施成本。
  - **需求变更管理：** 项目开发过程中可能出现的需求变更或范围扩展，若未能有效控制，将导致额外的工作量和成本增加，因此预留的风险金十分必要。

# Features
1. 已支持chrome浏览器
2. 可自定义目录文字颜色
3. 增加层级树功能，点击折叠符号可展开/折叠层级目录
4. 各目录添加h1/h2/h3...视觉提示
5. 点击目录可跳转到当前目录
6. 点击插件图片可显示/隐藏卡片
